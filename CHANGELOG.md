# Changelog

All notable changes to PackMatrix will be documented in this file.

This project currently follows a lightweight changelog format. A formal versioning strategy can be added before public release.

## Unreleased

### Added

- SwiftUI app for iPhone and Mac
- SwiftData local persistence
- Master packing list grouped by category
- Sample data seeding for starter categories and items
- Add, edit, delete, and quick-add flows for packing items
- Global duplicate prevention for packing item names
- Trip creation with destination, date range, and trip type
- Rule-based packing checklist generation
- Extra items during trip creation
- Trip checklist grouped by category
- Packed progress summary and hide-packed toggle
- Still To Pack section
- Recently packed suggestions
- Packing templates
- Trip duplication
- Trip archive and restore
- Trip deletion with confirmation and undo support
- Toast notification system
- Statistics screen
- Debug/health screen
- Custom PackMatrix app icon

### Changed

- Improved Create Trip date selection feedback
- Improved Quick Add toast feedback
- Improved checklist user experience and summary display

### Fixed

- Duplicate checklist item prevention
- Duplicate master item prevention
- Safe sample data seeding without overwriting user-created data
- Category picker reliability
- Basic validation for item and trip forms
